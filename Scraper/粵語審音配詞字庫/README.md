## apkg 卡片内容

每张卡片含有以下内容：

* `Id`: 解决 Anki 首字段不得重复的问题
* `Ranking`: 字的频序，频次越大排名越前
* `Freq.`: 字的频次，越大说明越常用
* `Characters`: 字
* `Syllable`: 音节（粤拼）
* `Words(Explanation) / Remarks`: 词汇等
* `Sound`: 读音文件；陳京英老师录音，故在尾部加上`_by_can4_ging1_ying1`
